# Perlin Noise Workshop

パーリンノイズでグリッドを遊び倒すワークショップ課題です。ノイズ値から明るさ・色・向き・文字・図形を切り替えながら、フィールドの見え方を比較できます。

## OpenProcessing へのコピー

OpenProcessing に移植する際は以下の順でファイルを読み込んでください。

1. `../common/utils.js` — 共有ユーティリティ（HUD やステータスが不要なら省略可）
2. `sketch.js` — p5.js スケッチ本体（`example.js` と同一内容）

### 手順メモ

1. OpenProcessing で新規スケッチを作成
2. Files タブに `sketch.js` を追加（必要なら `utils.js` も追加）
3. Run を押して動作確認

## ローカルプレビュー

```bash
npm run dev

# ブラウザで参照（開発者向け）
http://localhost:4173/dev/03_perlin_noise/local.html
```

## モード一覧

1. **BW** — 明度だけをノイズで変えるタイル表示
2. **HSB** — 色相/彩度/明度を別ノイズでコントロール
3. **Vector** — ベクターフィールド（矢印アニメ）
4. **ASCII** — ノイズ値を文字セットへマッピング（`C` でセット切替）
5. **Shapes** — 図形の種類・角度・サイズをノイズで変化

## キーボード操作

- `1`〜`5` : 表示モード切り替え
- `[` / `]` : グリッド密度を粗く／細かく
- `N` / `M` : ノイズスケールを拡大／縮小
- `V` / `B` : アニメ速度を速く／遅く
- `O` : 枠線の ON/OFF
- `G` : 補助グリッドの ON/OFF
- `H` : HUD 表示の ON/OFF
- `C` : ASCII セット切り替え
- `R` : 新しい乱数シードを取得（即リセット）
- `Space` : アニメーション一時停止／再開
- `S` : 画像を保存 (`noise_field.png`)

## 共有ヒント

- HUD が表示されない場合は `H` キーで再表示できます。
- ASCII セットを増やしたい場合は `sketch.js` 冒頭の `ASCII_SETS` に追記します。
- グリッド密度の上下限は `sketch.js` 内の `max(6, …)` / `min(200, …)` を調整すると素早く変更できます。
