# CCBT ひらめく道場 - アルゴリズムで絵を描こう

プログラミング入門ワークショップの教材です。自然界のアルゴリズムをP5.jsで再現しながら、プログラミングの基礎とアルゴリズム的思考を学びます。

## 🎨 スケッチ一覧

### 1. L-System (リンデンマイヤー・システム)
**sketchs/01_lsystem/**

木の成長パターンをシンプルなルールで再現します。
- 文字列書き換えによるフラクタル生成
- 再帰的な枝の描画
- パラメータ調整による多様な表現

### 2. ボロノイ図
**sketchs/02_voronoi/**

キリンの模様や細胞のパターンを「最も近い点」で分割して表現します。
- 距離計算による領域分割
- インタラクティブな点の追加
- 様々な色モード

### 3. パーリンノイズ
**sketchs/03_perlin_noise/**

雲、地形、波など自然界の滑らかなランダムさを表現します。
- ノイズ関数の活用
- 地形・雲・木目など多様なパターン
- アニメーション対応

### 4. セルオートマトン
**sketchs/04_cellular_automaton/**

貝殻の模様をシンプルなルールから創発させます。
- 1次元セルオートマトン
- 256通りのルールパターン
- 複雑さの創発体験

## 🚀 セットアップ

### 開発サーバーの起動

```bash
# 依存関係のインストール
npm install

# 開発サーバーを起動
npm run serve
# または
npm run dev
```

ブラウザで `http://localhost:8080` を開きます。

### 静的ビルドの生成

```bash
npm run build
```

ルートディレクトリの `dist/` に、静的配信に必要なファイル（トップページ、各スケッチ、開発者向けプレビューなど）がコピーされます。

### スクリーンショットテスト

各スケッチの動作確認とデバッグ用にスクリーンショットを自動撮影できます。

```bash
# Playwrightのインストール（初回のみ）
npm install -D playwright
npx playwright install chromium

# スクリーンショット撮影を実行
npm run test:screenshots
```

スクリーンショットは `screenshots/` ディレクトリに保存されます。

## 🎓 ワークショップの流れ

### 第1部: 木を描こう（約1時間）
1. 手で木を描いてみる
2. L-Systemのルールを手で実行
3. コードで木を描き、花を咲かせる

### 第2部: 自然のアルゴリズム（約1-2時間）
1. ボロノイ図でキリンの模様
2. パーリンノイズで雲や地形
3. セルオートマトンで貝殻の模様

### 第3部: 自由制作
好きなアルゴリズムを選んで、自分なりの表現を模索

## 🔧 技術スタック

- **P5.js**: クリエイティブコーディング用JavaScriptライブラリ
- **D3-Delaunay**: ボロノイ図の高速計算
- **Playwright**: スクリーンショット自動撮影
